Документація

Як користуватися CheckMe

Перевірка посилань

Щоб перевірити посилання на наявність загроз:

  1. Перейдіть на вкладку "Перевірка посилання"
  2. Введіть URL у текстове поле
  3. Натисніть кнопку "Перевірити посилання"
  4. Очікуйте на результати аналізу

Перевірка файлів

Щоб перевірити файл на наявність вірусів:

  1. Перейдіть на вкладку "Перевірка файлу"
  2. Натисніть на область для завантаження або перетягніть файл
  3. Оберіть файл для перевірки (максимальний розмір: 100 МБ)
  4. Натисніть кнопку "Перевірити файл"
  5. Очікуйте на результати аналізу

Типи загроз, які ми виявляємо

Для посилань

  • Фішингові сайти
  • Сайти з шкідливим ПЗ
  • Підозрілі перенаправлення
  • Сайти з небезпечним контентом
  • Шахрайські ресурси

Для файлів

  • Віруси та трояни
  • Шпигунське ПЗ
  • Рекламне ПЗ (Adware)
  • Шкідливі скрипти
  • Підозрілі виконувані файли

Про нас

Наша місія

CheckMe - це безкоштовний сервіс, створений для забезпечення безпеки користувачів в інтернеті. Ми прагнемо зробити цифровий простір безпечнішим для всіх, надаючи зручні інструменти для перевірки посилань та файлів.

Наша команда

Над розробкою CheckMe працює команда фахівців з кібербезпеки, які мають багаторічний досвід у виявленні та нейтралізації різноманітних загроз. Ми постійно вдосконалюємо наші алгоритми для забезпечення максимальної ефективності.

Наші принципи

  • Безкоштовність: Ми віримо, що безпека в інтернеті має бути доступною для всіх
  • Конфіденційність: Ми не збираємо та не зберігаємо особисту інформацію користувачів
  • Прозорість: Ми надаємо детальну інформацію про виявлені загрози
  • Постійне вдосконалення: Ми регулярно оновлюємо нашу базу загроз та алгоритми аналізу
